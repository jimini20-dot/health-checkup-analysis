# 데이터 설명서

> ⚠️ 원자료(`건강검진_실습데이터.xlsx`)는 보안상 이 저장소에 포함하지 않습니다.
> 실습 데이터는 강의 중 별도 안내된 Google Drive 경로에서 받습니다.

## 변수 정의

### 연속형 변수
| 변수 | 설명 | 단위 |
|------|------|------|
| age | 나이 | 세 |
| bmi | 체질량지수 | kg/m² |
| waist | 허리둘레 | cm |
| sbp | 수축기 혈압 | mmHg |
| glucose | 공복 혈당 | mg/dL |
| tg | 중성지방 | mg/dL |
| hdl | HDL 콜레스테롤 | mg/dL |
| crp | C-반응성 단백 | mg/L |

### 범주형 변수
| 변수 | 설명 | 값 |
|------|------|------|
| sex | 성별 | 남 / 여 |
| region | 지역 | (지역 코드) |
| smoking | 흡연 여부 | 흡연 / 비흡연 |
| exercise_freq | 운동 빈도 | (빈도 범주) |
| metabolic_syndrome | 대사증후군 | 0 / 1 |

## 결측 처리

- `hdl`, `crp`에 결측 존재 → **중앙값 대치**로 통일 (STEP 1 참고).
- 원본은 `df_raw`로 보존하여 대치 전후 비교 가능.
